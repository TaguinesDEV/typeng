# Typing Master

A Kivy-based typing game with account login and ranking.

## Features

- Create or log in to an account
- Track best score per user
- Save typing history and top rankings
- Simple practice game with timed typing phrases

## Run

1. Install dependencies:

```powershell
python -m pip install -r requirements.txt
```

2. Start the app:

```powershell
python main.py
```

## Notes

- User accounts are stored in `accounts.json`.
- Rankings are sorted by best score.
- The game gives you 60 seconds to type as many phrases as possible.
